\# Library Management System



This Odoo module is designed to simplify daily library operations, such as managing book inventories, member registrations, and the book issuing/returning process.



\## Key Features:

\* \*\*Book Management:\*\* Store detailed info like Title, ISBN, Author, and Publisher.

\* \*\*Categories:\*\* Organize books by different genres or topics.

\* \*\*Member Records:\*\* Keep track of library members and their contact info.

\* \*\*Smart Workflow:\*\* Automatically checks if a book is "In Stock" before issuing.

\* \*\*Auto-Update:\*\* Inventory levels (available copies) update automatically upon return.

\* \*\*Reporting:\*\* Visual graphs and list views to track library stats.



\## How to Install:

1\. Copy the `library\_management` folder into your Odoo \*\*addons\*\* directory.

2\. Restart your Odoo server.

3\. Log in to Odoo and activate \*\*Developer Mode\*\* from Settings.

4\. Go to the \*\*Apps\*\* menu and click \*\*'Update Apps List'\*\*.

5\. Search for "Library Management" and click \*\*Install\*\*.



\## Technical Details:

\* \*\*Diagrams:\*\* Check the `/doc` folder for the ERD, Use Case, and Activity diagrams.

\* \*\*Demo Data:\*\* Some sample books and members are included in the `/data` folder for quick testing.

\* \*\*Access Rights:\*\* Different permissions are set for Librarians (Admin) and Members (User) via CSV security files.
